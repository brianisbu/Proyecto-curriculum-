Habilidades

Uso de html


HTML es un lenguaje de marcado (no de programacion), que ayuda a la estructuracion de la pagina web.
Tambien si se utiliza con buenas practicas, podremos hacer que el sitio web tenga un mayor alcanze de usuarios

Uso de css


CSS es un lenguaje para darle estilos a las estructuras de HTML (tampoco es un lenguaje de programacion). Tambien lo usamos para dale direccion y formas a los eslemento HTML, y hacer que la pagina web se vea mas atractiva para el usuario

Uso de js


JavaScript es un lenguaje de programacion (uno de los mas populares en la actualidad), que permite darle funcionalidades a la pagina web. Por ejemplo tenemos la seccion de Habilidades, que al hacer click se habren las pestañas de informacion de cada lenguaje, o al hacer scroll, los elementos aparecen con una animacion. Esto esta logrado gracias a la combinacion de HTML,CSS, y JavaScript

Uso de bootstrap


Bootstrap es un framework (tambien dicho en español libreria) front-end utilizado para desarrollar paginas web responsivas, osea, adaptables a diferentes interfaces.
Esta libreria ayuda a crear la estetica de cada sitio web muchisimo mas rapido gracias a sus componentes predeterminados. Esos componentes se pueden adaptar a las necesidades del cliente
